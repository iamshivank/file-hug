# File Hug 🫂

**Never lose something you wanted to remember.**

File Hug is an AI-powered memory system for everything you discover online.

Save reels, articles, tweets, ChatGPT conversations, links, notes, ideas, and resources in one place. Instead of creating folders or bookmarks you'll never revisit, File Hug lets you save information instantly and find it later using natural language.

---

## The Problem

Every day we discover valuable content:

* Instagram reels we want to revisit
* YouTube videos worth watching later
* Articles we intend to read
* ChatGPT conversations with useful insights
* Business ideas, notes, and random thoughts

Most of it gets lost across bookmarks, screenshots, notes apps, browser tabs, and social media saves.

Months later, we remember the idea—but not where we saved it.

---

## The Solution

File Hug becomes a second brain for your digital discoveries.

Save anything in seconds and retrieve it later simply by asking:

* "Show me the startup idea I saved about AI tutors."
* "Find that Instagram reel about productivity."
* "What was that article about building habits?"
* "Show my notes related to SaaS pricing."

No folders. No manual organization. Just save and search.

---

## Current Features

### Save Links

Store content from platforms such as:

* Instagram
* YouTube
* X (Twitter)
* TikTok
* Reddit
* GitHub
* Medium
* Any website

File Hug automatically identifies the platform and content type.

### Save Notes

Create quick notes, ideas, reminders, and thoughts alongside saved links.

### Smart Organization

Content is automatically categorized using platform and content-type detection.

Examples:

* Instagram Reel
* YouTube Short
* GitHub Repository
* Reddit Post
* Personal Note

### Unified Memory Feed

View all your saved content in one searchable timeline.

---

## How It Works

1. Save a link or note.
2. File Hug automatically understands what it is.
3. Everything is stored in a personal memory library.
4. Search later using natural language.
